# Icon Replacement Instructions

To use the new purple icon:

1. Download or save the purple icon image (1024x1024px PNG recommended)
2. Replace the existing icon file:
   - Main icon: assets/icon.png (1024x1024px)
   - Adaptive icon: assets/adaptive-icon.png (1024x1024px for Android)
   - Splash icon: assets/splash-icon.png (1024x1024px)

3. The icon should be:
   - Square (1024x1024px minimum)
   - PNG format
   - Purple color (#9C27B0 or similar)
   - Transparent background recommended

After replacing the icon files, run the build command.
