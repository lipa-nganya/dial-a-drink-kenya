# Valkyrie Console

Partner-facing web dashboard for DeliveryOS Valkyrie Partner API.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The console will run on **http://localhost:3002**

## Features

- **Partner Login**: Email/password or API key authentication
- **Overview Dashboard**: Active deliveries, completed orders, revenue metrics
- **Order Management**: Create orders, view order status, request driver assignment
- **Driver Fleet**: View partner-owned drivers, activate/deactivate drivers
- **Billing & Usage**: View completed deliveries and revenue

## Environment Variables

- `REACT_APP_VALKYRIE_API_URL`: Backend API URL (defaults to localhost:5001/api/valkyrie/v1)
